# EveryNight

#### 这是一款阅读类app，每夜推送一篇美文作为睡前读物，重拾阅读的魅力。

##注意
先运行server文件夹下s.js服务器

用命令提示符 node s.js

出现Server running at http://localhost:18000/ 表示服务器开启成功

服务器开启成功后再打开app
